# Reproducibility checklist (ACM-style, adapted)

This checklist is designed for quick artifact evaluation.

## Artifact availability

- Source: repository checkout.
- Evidence artifacts: results/ (generated by `make reproduce-all`).

## Environment

- OS: Linux (CI).
- Python: 3.12.x primary (CI pins 3.12).

## Build / install

```sh
python3.12 -m venv .venv && . .venv/bin/activate
python -m pip install -U pip
python -m pip install -e ".[dev,plots]"
```

## Run

```sh
make reproduce-all
python -m pytest -q
```

## Verify

- Read artifact index: results/README.md
- Verify integrity metadata:
  - results/manifest.json
  - results/hashes.txt

## If results diverge

See docs/REPRODUCIBILITY.md.
